Visão Geral do Projeto

Este projeto apresenta um dashboard interativo completo desenvolvido em R e Shiny para análise abrangente de dados de vendas de e-commerce. A aplicação demonstra competências avançadas em análise de dados, visualização interativa, desenvolvimento web com Shiny, Business Intelligence e processamento de dados.

O objetivo é fornecer uma ferramenta prática para análise de vendas, implementar boas práticas de desenvolvimento e documentação, e oferecer insights acionáveis para a tomada de decisões empresariais.

Funcionalidades Principais

  • Dashboard Principal: KPIs dinâmicos (vendas totais, ticket médio, conversão), gráficos interativos, métricas de performance e alertas inteligentes.
  • Análise Temporal: Séries temporais de vendas, identificação de sazonalidade, análise de tendências e previsões.
  • Análise de Produtos: Performance por categoria, análise ABC, margem de lucro e ciclo de vida dos produtos.
  • Análise Geográfica: Vendas por região, mapas interativos e comparação entre regiões.
  • Análise de Clientes: Segmentação RFM, Lifetime Value, Churn Analysis e padrões de compra.
  • Relatórios e Exportação: Geração de relatórios em PDF, exportação de dados (CSV, Excel, JSON) e agendamento de relatórios.

Stack Tecnológico

O dashboard foi construído utilizando as seguintes tecnologias:

  • R 4.1+: Linguagem principal para análise estatística.
  • Shiny: Framework para aplicações web interativas.
  • shinydashboard: Interface de dashboard profissional.
  • plotly: Gráficos interativos e responsivos.
  • DT: Tabelas interativas com funcionalidades avançadas.
  • Outros pacotes R para manipulação de dados (dplyr, tidyr, lubridate), análise estatística (ggplot2, forecast, corrplot) e UI/UX (shinyWidgets, shinycssloaders).

Instalação e Uso

Para executar o dashboard localmente, siga os passos:

  1. Clone o repositório: git clone https://github.com/galafis/r-sales-dashboard-shiny.git
  2. Navegue até o diretório: cd r-sales-dashboard-shiny
  3. Instale as dependências R (se necessário): install.packages(c("shiny", "shinydashboard", "plotly", ...))
  4. Execute a aplicação no RStudio ou via console R: shiny::runApp()

Para mais detalhes, consulte o README.md do repositório.
